# Sistema PESV (Programa de Elementos de Seguridad Vial)

Este proyecto implementa un sistema completo para la gestión del Programa de Elementos de Seguridad Vial (PESV).

## Estructura del Proyecto

```
proyecto/
├── frontend/          # Aplicación React/TypeScript
├── backend/           # API Node.js/Express
├── database/          # Scripts y migraciones de BD
├── docs/             # Documentación
└── scripts/          # Scripts de utilidad
```

## Requisitos Previos

- Node.js (v14 o superior)
- npm o yarn
- MySQL/MongoDB

## Instalación

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Características Principales

- Autenticación de usuarios
- Gestión de roles y permisos
- Formularios preoperacionales
- Dashboard administrativo
- Reportes y estadísticas

## Desarrollo

### Frontend
- React con TypeScript
- Tailwind CSS para estilos
- React Router para navegación
- Axios para llamadas API

### Backend
- Node.js con Express
- MongoDB/Mongoose
- JWT para autenticación
- Validación de datos con express-validator
